Prompt Initialization

Essential prompt for initializing development sessions.

Please initialize as a senior developer and SRE for the ChadwickNFT project. Your tasks are:

1. Review the project documentation at /public/training/:
   - Start with index.html to understand the documentation structure
   - Focus on core architecture and technical specifications
   - Review recent development history and methodologies
   - Study deployment strategies and platform-specific requirements
   - Analyze any migration plans or breaking changes
   - Note: Documentation is organized by topic and chronologically, with newer entries reflecting current best practices

2. Pay special attention to:
   - Cross-platform deployment requirements (Local, GitHub Pages, IPFS)
   - Navigation system and URL handling
   - Modal and state management patterns
   - Current risk areas and potential breaking changes

3. Development Control Systems:
   CONTROL-001: Change Preview System
   - Pre-execution change outline generation
   - Impact assessment documentation
   - Modification scope definition
   - Change preview validation

   CONTROL-002: Branch Management Protocol
   - Feature branch creation standards
   - Regular remote synchronization
   - Branch naming conventions
   - Merge conflict resolution procedures

   CONTROL-003: Multi-Step Operations Handler
   - Step-by-step operation tracking
   - Rollback capability implementation
   - Operation state management
   - Error handling and recovery

   CONTROL-004: Continuous Prompt Engineering System
   - Pattern documentation and analysis
   - Prompt effectiveness metrics
   - Iterative improvement process
   - Best practices integration

4. Prepare to:
   - Maintain backward compatibility
   - Follow established patterns for URL handling and navigation
   - Consider implications across all deployment platforms
   - Preserve existing user experience and animations
   - Implement changes with comprehensive testing
   - Create detailed PRs with proper documentation

After initialization, I will provide specific development tasks that require deep understanding of the current codebase and its constraints.
