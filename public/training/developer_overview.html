ChadwickNFT Developer Overview

1. Tech Stack

Core Technologies

  • React 18.3.1 (with Hooks)
  • React Router 6.22.2 (HashRouter for multi-platform compatibility)
  • Tailwind CSS 3.4.1 (with custom configuration)
  • Framer Motion (for animations)

Build Tools

  • Create React App (not ejected)
  • PostCSS with Autoprefixer
  • Babel with custom plugins for modern JavaScript features

2. Component Architecture

Core Components

  • App.js: Root component with routing setup
  • Navbar.js: Responsive navigation with mobile menu
  • Home.js: Landing page with hero section
  • Gallery.js: Portfolio grid with masonry layout
  • ImageCarousel.js: Full-screen image viewer with modal
  • BWGallery.js: Black and white photography section
  • About.js: Artist information page
  • Contact.js: Contact form and social links

Utility Components

  • utils/urlUtils.js: Cross-platform URL handling

3. Styling Architecture

Tailwind Configuration

  • Custom color palette with NFT-themed colors
  • Extended animation configurations
  • Custom breakpoints for responsive design
  • Typography system with custom font stack

Animation System

  • Framer Motion for page transitions
  • Custom hover effects
  • Smooth modal animations
  • Scroll-based navbar transformations

4. State Management

  • React Hooks for local state management
  • URL-based routing state
  • Component-specific state patterns:
    • Image carousel state for current image and modal
    • Navigation state for mobile menu and scroll position
    • Gallery state for masonry layout and image loading

5. Development Workflow

Local Development

  • Start development server: npm start
  • Build for production: npm run build
  • Platform-specific builds:
    • GitHub Pages: npm run build:github
    • IPFS/Fleek: npm run build:fleek

Code Organization

  • src/components/: React components
  • src/utils/: Utility functions
  • public/images/: Asset directory
    • bw/: Black and white photography
    • thumbnails/: Optimized preview images
    • large/: Full-size images

6. Deployment

  • Multi-platform deployment strategy
  • GitHub Actions automation
  • Environment-specific configurations
  • Cross-platform URL handling

7. Best Practices

  • Component composition over inheritance
  • Custom hooks for reusable logic
  • Responsive design first approach
  • Progressive enhancement
  • Performance optimization:
    • Image optimization pipeline
    • Code splitting
    • Lazy loading
